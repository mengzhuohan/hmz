1. Git 和 SVN 两款工具的主要区别是什么？
     Git是分布式，SVN是集中式
2. 我们使用的 Git 客户端工具是什么？
     TortoiseGit-2.10.0.2-64bit（小乌龟）
3. 当前主流的 Git 代码托管平台有哪些，分别属于哪个公司？
     GitHub                   微软
     开源中国Gitee         深圳市奥思网络科技有限公司
4. 对于远程仓库 pull 和 clone 命令的区别是什么？
     pull是有本地仓库时，将远程仓库里新的commit数据下载过来，并且与本地仓库合并
     clone是没有本地仓库时，将远程仓库整个下载过来
5. 说明本地工作目录、暂存区和本地仓库的区别和作用分别是什么？
     工作目录：平时存放代码的地方     
     暂存区：临时存放改动
     本地仓库：安全存放数据的位置
     在工作目录中添加、修改文件，将需要进行版本管理的文件放入暂存区，将暂存区的文件提交到本地仓库
6. 说明如何进行版本的回退？
     重置（reset）
     重置当前分支到某个提交的版本（版本号）
     硬重置：将暂存区和本地工作目录中的内容完全回退到指定的提交版本
7. 访问远程 Gitee 仓库有几种途径，请简述说明。
     本地仓库与远程仓库建立源连接
     创建电脑的公钥私钥（SSH）
     提交本地代码到远程仓库
8. 简述在 Gitee 平台上，分支、标记的作用。
     分支：是由指针管理起来的，适合大型项目的开发。在分支上做开发，调试好了后再合并到主分支。
     那么每个人开发模块式都不会影响到别人。
     标记作用：在开发的一些关键时期,使用标签来记录这些关键时刻, 例如发布版本, 有重大修改, 升级的时候, 
     会使用标签记录这些时刻, 来永久标记项目中的关键历史时刻;
